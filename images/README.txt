วางไฟล์รูปสื่ออินโฟกราฟิก 2 ไฟล์ในโฟลเดอร์นี้:

  group-a.jpg   →  สื่อ "แบบเดิม"  (ผู้ตอบกลุ่ม A จะเห็นรูปนี้)
  group-b.jpg   →  สื่อ "แบบใหม่"  (ผู้ตอบกลุ่ม B จะเห็นรูปนี้)

ข้อแนะนำ:
- ใช้นามสกุล .jpg หรือ .png ก็ได้ (ถ้าใช้ .png ให้แก้ชื่อใน index.html → CONFIG.IMAGE_A / IMAGE_B)
- ควรเป็นภาพแนวตั้ง ความกว้างอย่างน้อย 1000 px เพื่อให้คมชัดบนมือถือ
- ขนาดไฟล์ไม่ควรเกิน ~1–2 MB เพื่อให้โหลดทันใน 15 วินาที

หากต้องการเปลี่ยนชื่อไฟล์หรือใช้ลิงก์รูปจากอินเทอร์เน็ต
ให้แก้ที่ส่วน CONFIG ด้านบนของไฟล์ index.html
